哨兵一号 Sentinel-1 GRD 数据简介

C波段雷达影像（5.405GHz），可全天时、全天候成像

什么是 GRD 数据？

GRD（Ground Range Detected）是 Sentinel-1 雷达的标准产品，已处理为辐射定标 + 地形校正后的后向散射系数，单位为 dB。

像素值表示地物对雷达波的反射强弱，可用于识别水体、城市、农田等。

可以用来做什么？

✅ 水体监测 / 洪涝提取
✅ 农作物淹水期 / 种植结构变化
✅ 城市扩张 / 基础设施监测
✅ 海洋船只 / 油膜探测
✅ 变化检测（多时相对比）

不能用来做什么？

❌ 不能用于干涉测量（InSAR）或形变分析（因为 GRD 已去除相位信息）
❌ 无法生成高精度地形（需使用 SLC 产品）
❌ 无法表示真实颜色或植被类型（非光学影像）
❌ 噪声较大，需要滤波 / 统计分析后使用

成像模式（Instrument Mode）

  • IW 模式 = “扫描式拍摄”：雷达天线边飞边左右摆动，把多个条带拼成一张大图，清晰度约 10 米，是陆地监测的默认模式。
  • EW 模式 = “超宽角度拍摄”：摆动得更大，换来更广视野，但画质下降到约 25~40 米，常用于海洋观测。
  • SM 模式 = “直直地拍”：不摆头，画质最好，但范围小，普通用户很少接触到，在 Earth Engine 中极少出现。

轨道方向（Orbit Pass）

  • 🟦 ASCENDING（升轨） — 南向北飞行
  • 🟥 DESCENDING（降轨） — 北向南飞行

为何需要区分升轨 / 降轨？

  • 观测方向不同：SAR 雷达成像是侧视模式，升轨与降轨会从不同方位照射地物，导致同一地物在 VV/VH 波段下的亮度差异很大。
  • 成像时间不同：降轨通常在上午，升轨在下午，光学影像的阴影长度和亮温都会受影响。
  • 避免混用导致误判：进行变化检测时（如地表沉降、水体监测），必须使用相同轨道方向的数据，否则会出现“假变化”。
  • 由于 SAR 的侧视成像特性，升轨与降轨从不同侧面照射地形，例如升轨使山的西坡信号增强，而降轨则使东坡更亮，因此两者不能直接拼接或用于合成，否则会产生明显的不一致。

极化方式（Polarization）

  • VV：垂直发射 / 垂直接收（最主流）
  • VH：垂直发射 / 水平接收（常与 VV 配对）
  • HH：水平发射 / 水平接收（多见于海洋）
  • HV：水平发射 / 垂直接收（较少）

GRD中可选的极化波段组合

Sentinel-1 SAR 的极化方式决定了雷达波的发射和接收方向，不同极化对地物的反射特性敏感度不同。GRD 数据中只提供以下四种极化组合：

  • VV：单极化，垂直发射 / 垂直接收（最常用，用于陆地监测）
  • HH：单极化，水平发射 / 水平接收（主要用于海洋和船舶探测）
  • VV + VH：双极化，垂直发射 / 垂直+水平接收（VV 为共极化，VH 为交叉极化，常用于植被和农田监测）
  • HH + HV：双极化，水平发射 / 水平+垂直接收（HH 为共极化，HV 为交叉极化，较少使用）

✅ Sentinel-1 GRD 数据之所以只提供这四种组合，是因为它们覆盖了绝大部分地表信息，同时兼顾数据量和处理复杂度。额外极化组合虽然可能提供更多信息，但对常规地表监测增益有限，因此未在 GRD 产品中提供。
