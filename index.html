哨兵一号 Sentinel-1 GRD 数据简介

免费的 C 波段雷达影像，可全天时、全天候成像

什么是 GRD 数据？

GRD（Ground Range Detected）是 Sentinel-1 雷达卫星提供的一种标准产品，经过正射校正和幅度计算，适合快速可视化和一般分析。

特点：已进行辐射定标 + 地理编码，像素值为后向散射强度（dB）。

GRD 数据可以用来做什么？

  • 水体提取 / 洪涝监测
  • 农作物变化监测（如水稻种植淹水期）
  • 城市扩张 / 建筑物识别
  • 地表运动（粗略）/ 山体滑坡前后变化
  • 海洋油膜、船只探测

GRD 数据不能做什么？

  • 不能直接用于高精度地形测量（需要 SLC 干涉产品）
  • 不能反映真实地表颜色或植被类型（是雷达信号，不是光学影像）
  • 无法区分细粒度地物（如小型道路、树种等）
  • 噪声较大，需经过滤波和区域统计后才能用于机器学习

常见极化模式

  • VV / VH（最常见）：垂直发射+垂直/水平接收
  • HH / HV（较少）：多见于海洋或极地区域
  • 多数陆地地区仅提供 VV 或 VV+VH 数据
